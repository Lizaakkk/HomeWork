# Крестики-нолики (Tic-Tac-Toe)


Консольная реализация классической игры "Крестики-нолики" на Python. 

Как играть
Игровое поле представляет собой сетку 3x3 с координатами:

text
    | 0 | 1 | 2 |
------------------
  0 |   |   |   |
------------------
  1 |   |   |   |
------------------
  2 |   |   |   |
------------------
Игроки по очереди вводят координаты хода в формате: строка столбец

Например: 1 1 для центральной клетки

Первый игрок использует X, второй - O

Побеждает игрок, первым собравший линию из трех своих символов
